# Linux Distros List (Personal Project)

A simple android application containing the information of some of the popular linux distros. Created using kotlin to learn and practice about activity, intent, view, style, and RecyclerView.

Created to complete the course of 
["Belajar Membuat Aplikasi Android untuk Pemula"](https://www.dicoding.com/certificates/MRZMQVJO0PYQ) from Dicoding.
